# obs_monocam
A repository to hold the monocam description.
Monocam is a prototype LSST CCD; data taken at NOFS May 2016.

This package was used in the writing of
_Fringing in MonoCam Y4 filter images_, J. Brooksa, M. Fisher-Levineb and A. Nomerotskia (2017)
(<https://iopscience.iop.org/article/10.1088/1748-0221/12/05/C05003/>)
